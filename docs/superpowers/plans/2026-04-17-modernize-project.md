This is a transaction import automation tool for YNAB (You Need A Budget). It reads transactions from multiple bank CSV exports (Swedbank, Revolut, TBC) and imports them into YNAB via the official API.

## Architecture

### Provider Pattern
Each bank provider (Swedbank, TBC, Revolut) follows the same pattern:
1. **Models** (`models.py`) — CSV dataclass definitions
2. **Providers** (`providers.py`) — Reader class implementing `BaseProvider`
3. **Transformers** (if needed) — Transform to common `Transaction` model

### Transaction Flow
```
CSV File → Bank Provider.read() → [Transaction, ...] → YNAB API.write() → Success
```

- `.env` file for API keys and paths (see README)
- Pydantic `Settings` class in `settings.py` for validation
- Environment variables: `YNAB_API_KEY`, `YNAB_BUDGET_ID`, `YNAB_ACCOUNT_ID`, `READ_FROM_PATH`, `READER`, `WRITER`

## Type Hints: Before & After

❌ **Before:**
```python
def read_transactions(path):
    data = []
    for row in csv.reader(open(path)):

✅ **After:**
```python
def read_transactions(path: str) -> List[Transaction]:
    transactions: List[Transaction] = []
    with open(path) as f:
        for row in csv.reader(f):
            transactions.append(Transaction(...))
    return transactions
